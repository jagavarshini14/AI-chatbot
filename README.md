# AI Chatbot using NLTK

This is a simple rule-based chatbot built using Python and the nltk library.

## 👤 Created by
*Varsh*

## 📁 Files
- chatbot.py : Main chatbot code

## ⚙ How to Run

1. Clone the repo or download the files
2. Install NLTK:

pip install nltk

3. Run the chatbot:

python chatbot.py
